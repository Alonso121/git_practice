Css Properties Cheatsheet

Common Css Properties
No. Property Description Value ex.
1 color: Changes the text color of selected tag midnightblue;
2 background-color: Changes the text bakground color to be used lightbluesky;
3 font-family: The CSS font-family property defines the font to be used. "Arial Black";
4 font-size: The CSS font-size property defines the text size to be used. 22px;
5 text-align: Changes the alignment of text of selected element center;

Table Properties CSS
No. Property Description Value ex.
1 border: To specify table borders in CSS 1px solid black;
2 border-collapse: Collapses table borders into single border collapse;
3 width: Sets the width of the table in % of the page size or in px 100%;
4 height: Sets the total height of the table or table elements 60px;
5 padding: Controls the space between the border and text in table 15px;
